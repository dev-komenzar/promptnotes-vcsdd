### Rust (`editor_wire_sprint8.rs`)

```
error[E0432]: unresolved imports `promptnotes_lib::editor::BlockTypeDto`,
  `promptnotes_lib::editor::DtoBlock`, `promptnotes_lib::editor::PendingNextFocusDto`,
  `promptnotes_lib::editor::compose_state_for_cancel_switch`, ...
```

the new Sprint-8 `enum` variants (`Idle`, `Editing { ... }`, etc.) and the `compose_*`
pure constructors do not exist yet.

The fixture is produced by `cargo test print_wire_fixtures` (Phase 2b).

undocumented until Phase 2b replaces the todo with a full assertion.
